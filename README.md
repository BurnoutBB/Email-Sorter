## Email Filter Tool

This Python script filters a list of email addresses and passwords combolists based on a specified domain.

## How to Use

    -git clone https://github.com/BurnoutBB/Email-Sorter
    -cd Email-Sorter
    -python email-sorter.py

## Requirements

- Python 3.x
